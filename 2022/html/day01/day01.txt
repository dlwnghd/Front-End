HTML
	HyperText Markup Language
	텍스트로 페이지의 구조와 데이터를 표기하는 언어
	HTML 문서는 확장자가 .html이다.
	HTML 문서는 문서를 작성할 수 있는 프로그램이라면 어디서든지
	작성이 가능하다.
	HTML 문서는 대소문자를 구별하지 않는다.
	HTML 문서는 띄어쓰기, 줄바꿈을 구별하지 않는다.
	HTML 문서는 태그 형태로 괄호(<명령어>)를 사용하여 표현한다.
	
HTML의 요소(태그)
	①<p>③첫 개강 ②</p>

	① 여는 태그	: 요소의 이름(p)과 열고 닫는 꺽쇠괄호로 구성
	② 닫는 태그	: 요소의 이름 앞에 슬래쉬(/)가 있다.
	③ 내용(Content)	: 요소의 내용이며, 단순한 텍스트이다.

	1~3번을 통틀어서 요소라고 한다.
	요소의 이름은 가독성을 위해서 소문자로 작성하는 것을 권장한다.
